About CyberHub

CyberHub is a cutting-edge demo application fortified by a custom-built WAF, simulating a production-grade web environment. It enables users to safely explore secure routes, test attack vectors (e.g., SQL injection, XSS), and observe real-time defense mechanisms in action. Ideal for learners, security professionals, and developers.

Technical Specifications

  • Framework: Built on Flask with Python for scalability and performance.
  • WAF Engine: Custom rule-based system with real-time threat detection.
  • Security Features: Protection against SQLi, XSS, path traversal, and more.
  • Monitoring: Real-time request logging and server uptime tracking.
  • Scalability: Supports up to 50 recent request logs with FIFO queueing.

Note: Specifications subject to updates as CyberHub evolves.
